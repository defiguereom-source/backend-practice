## Conceptos aprendidos

### Java Básico

Java es un lenguaje de programación orientado a objetos, ampliamente utilizado en desarrollo backend, aplicaciones empresariales y sistemas robustos.

- Lenguaje compilado  
- Tipado estático  
- Multiplataforma (JVM)  
- Enfoque en programación orientada a objetos
