## Conceptos aprendidos

### Programación Orientada a Objetos

La POO es un paradigma de programación basado en la creación de objetos que representan entidades del mundo real.

#### Pilares principales:

- **Encapsulamiento**: protección de datos dentro de clases  
- **Herencia**: reutilización de código entre clases  
- **Polimorfismo**: múltiples comportamientos para un mismo método  
- **Abstracción**: simplificación de la complejidad del sistema
