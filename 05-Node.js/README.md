## Conceptos aprendidos

### Node.js

Node.js es un entorno de ejecución de JavaScript en el servidor que permite construir aplicaciones backend rápidas y escalables.

- Basado en eventos  
- Asíncrono y no bloqueante  
- Ideal para APIs y servicios web  
- Usa el ecosistema de npm  

---

## APIs REST

Con Node.js se pueden crear APIs que permiten la comunicación entre frontend y backend.

- Métodos HTTP (GET, POST, PUT, DELETE)  
- Respuestas en formato JSON  
- Manejo de rutas  
- Control de solicitudes y respuestas  

---

## Express.js

Framework utilizado para simplificar la creación de servidores en Node.js.

- Rutas organizadas  
- Middleware  
- Manejo de peticiones HTTP  
- Creación rápida de APIs
